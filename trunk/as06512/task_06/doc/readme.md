## Ход выполнения

__Изучение предложенных открытых проектов__

В ходе выполнения лабораторной работы изучил предложенные открытые проекты. Вкрадце: 

PLCnext-howto — практическая документация по архитектуре, настройке и интеграции PLCnext с внешними системами (OPC UA, Node-RED). Позволяет быстро начать разработку.

ptusa-main — базовая программа для контроллеров PLCnext. Изучена её структура, механизмы работы со скриптами и промышленными протоколами. Служит каркасом для прикладных решений.

EPLANner — дополнение для EPLAN Electric P8. Автоматизирует создание документации и экспорт конфигураций в PLCnext, ускоряя перенос данных из схем в программный проект.

T1-PLCnext-Demo — демо-проект с готовой структурой, скриптами и настройками. Может использоваться как шаблон для старта собственной разработки.
