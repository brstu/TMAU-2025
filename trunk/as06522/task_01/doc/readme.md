## Задание 1. Моделирование управляемого объекта

В качестве объекта управления рассматривается температурный режим, описываемый дифференциальным уравнением:

$$\Large\frac{dy(\tau)}{d\tau}=\frac{u(\tau)}{C}+\frac{Y_0-y(\tau)}{RC} $$ (1)

где $\tau$ - время; $y(\tau)$ - температура объекта; $u(\tau)$ - подводимая тепловая мощность; $Y_0$ - температура окружающей среды; $C, RC$ - постоянные коэффициенты.

После дискретизации получены линейная (2) и нелинейная (3) модели объекта:

$$\Large y_{\tau+1}=ay_{\tau}+bu_{\tau}$$ (2)

$$\Large y_{\tau+1}=ay_{\tau}-by_{\tau-1}^2+cu_{\tau}+d\sin(u_{\tau-1})$$ (3)

где $\tau$ - дискретные моменты времени ($1,2,3{\dots}n$); $a,b,c,d$ - постоянные коэффициенты.

Цель работы: разработать программу на языке **С++**, реализующую моделирование температуры объекта по обеим моделям.
