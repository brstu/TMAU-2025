# Лабораторная работа №1  
## Моделирование управляемого объекта

### Цель работы
Целью данной лабораторной работы является изучение принципов математического моделирования управляемого объекта и реализация программной модели, описывающей динамику изменения температуры во времени.

---

### Условия задания
Необходимо реализовать программу на языке **C++**, которая выполняет имитацию температурного процесса управляемого объекта в дискретные моменты времени.  
Результаты моделирования должны быть представлены для двух вариантов математической модели:
- линейной;
- нелинейной.

Исходный код программы размещается в каталоге `task_01/src`, а отчет по работе оформляется в формате `Markdown` и сохраняется в каталоге `task_01/doc`.

---

### Теоретические сведения
Рассматриваемый объект описывается уравнением теплового баланса, связывающим температуру объекта, управляющее воздействие и параметры среды.  
После дискретизации по времени исходная модель может быть представлена в виде рекуррентных соотношений.

**Линейная модель:**

\[
y_{t+1} = a \cdot y_t + b \cdot u_t
\]

**Нелинейная модель:**

\[
y_{t+1} = a \cdot y_t - b \cdot y_{t-1}^2 + c \cdot u_t + d \cdot \sin(u_{t-1})
\]

где  
- \( y_t \) — температура объекта в момент времени \( t \);  
- \( u_t \) — управляющее воздействие;  
- \( a, b, c, d \) — постоянные коэффициенты модели.

---

### Реализация
Программа реализует пошаговое вычисление температуры объекта для заданного количества временных шагов.  
Для каждого шага вычисляются новые значения температуры по линейной и нелинейной моделям, а также формируется управляющее воздействие.

В ходе работы используются:
- стандартные контейнеры STL;
- математические функции из стандартной библиотеки;
- форматированный вывод данных.

---

### Результаты
В результате выполнения программы формируется таблица значений, содержащая:
- номер временного шага;
- значение управляющего воздействия;
- температуру объекта для линейной модели;
- температуру объекта для нелинейной модели.

Это позволяет сравнить поведение двух моделей и проанализировать влияние нелинейных факторов.

---

### Вывод
В ходе лабораторной работы была разработана программная модель управляемого объекта.  
Проведено сравнение линейной и нелинейной моделей, показаны различия в их динамике и чувствительности к управляющему воздействию.  
Полученные результаты подтверждают необходимость учета нелинейных эффектов при моделировании реальных физических процессов.
