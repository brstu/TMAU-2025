# Лабораторная работа №2  
## Модульное тестирование программы

### Цель работы
Целью лабораторной работы является освоение принципов модульного тестирования программ на языке C++ с использованием фреймворка Google Test, а также проверка корректности работы основных функций программы, разработанной в лабораторной работе №1.

---

### Постановка задачи
В рамках данной работы необходимо
1. Модифицировать программу из лабораторной работы №1, выделив основные вычислительные функции.
2. Разработать модульные тесты для ключевых функций программы.
3. Использовать фреймворк Google Test для автоматического выполнения тестов.
4. Разместить исходный код программы в каталоге `task_02src`.
5. Разместить модульные тесты в каталоге `task_02test`.
6. Отразить факт выполнения работы и наличие тестов в файле `README.md`.

---

### Краткое описание программы
Программа моделирует поведение управляемого объекта с использованием дискретных математических моделей.  
Реализованы две модели
- линейная модель изменения состояния;
- нелинейная модель, учитывающая предыдущее состояние и управляющее воздействие.

Для повышения тестируемости логика вычислений вынесена в отдельные функции.

---

### Модульное тестирование
Для тестирования используется фреймворк Google Test.  
Модульными тестами проверяются
- корректность вычисления одного шага линейной модели;
- корректность вычисления одного шага нелинейной модели;
- сохранение размеров выходных данных при моделировании;
- корректная инициализация начальных условий;
- различие результатов линейной и нелинейной моделей при одинаковых входных данных.

Тесты выполняются автоматически в процессе сборки проекта.
