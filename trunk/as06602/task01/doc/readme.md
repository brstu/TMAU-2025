# Отчет по лабораторной работе №1  
## Моделирование объекта управления  

### 1. Цель работы  
Разработка программы на C++ для моделирования температуры объекта управления на основе линейной и нелинейной математических моделей, полученных из дифференциального уравнения теплового процесса.  

### 2. Теоретическая часть  
Исходное дифференциальное уравнение объекта:  
```
dy(τ)/dτ = u(τ)/C + (Y₀ - y(τ))/RC
```

После дискретизации получены две модели:  
**Линейная модель:**  
```
y_{τ+1} = a*y_τ + b*u_τ
```  

**Нелинейная модель:**  
```
y_{τ+1} = a*y_τ - b*y_{τ-1}² + c*u_τ + d*sin(u_{τ-1})
```

### 3. Реализация программы  
Программа состоит из:  
1. Класса `TemperatureModel` для моделирования объекта  
2. Функций расчета по линейной и нелинейной моделям  
3. Генератора управляющих воздействий  
4. Модуля вывода результатов  

**Параметры моделирования:**  
- Коэффициенты: a=0.98, b=0.05, c=0.03, d=0.02  
- Начальная температура: y₀ = 20.0°C  
- Температура помещения: Y₀ = 25.0°C  
- Время моделирования: 100 шагов  
- Шаг дискретизации: 1 условная единица времени  

**Структура проекта:**

### 4. Результаты моделирования  
#### 4.1. Ключевые характеристики:  
- **Линейная модель** демонстрирует монотонное приближение к температуре помещения  
- **Нелинейная модель** показывает колебательный характер переходного процесса  
- Обе модели в установившемся режиме стремятся к Y₀ = 25.0°C  
- Наибольшие расхождения между моделями наблюдаются в начальный период  

#### 4.2. Пример результатов (первые 5 шагов):  
```
Шаг    u(τ)      Линейная    Нелинейная
--------------------------------------
0      10.00      20.00        20.00
1      14.99      20.75        20.75
2      19.88      21.83        21.80
3      24.57      23.21        23.11
4      28.96      24.83        24.62
```

#### 4.3. Статистика (после 100 шагов):  
- Температура помещения (Y₀): 25.0°C  
- Установившаяся температура (линейная модель): 24.95°C  
- Установившаяся температура (нелинейная модель): 24.87°C  
- Разница между моделями: 0.08°C  

### 5. Анализ результатов  
1. **Сходимость моделей:** Обе модели демонстрируют устойчивое поведение и стремятся к температуре помещения  
2. **Влияние нелинейностей:** Квадратичный член и синусоидальная зависимость в нелинейной модели приводят к более сложной динамике  
3. **Чувствительность к управлению:** Обе модели адекватно реагируют на изменения управляющего воздействия  
4. **Вычислительная устойчивость:** Программа обеспечивает стабильные вычисления без расходимости
